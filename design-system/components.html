Components

The repeated parts. Every screen in the app is a recombination of these — buttons, stats, holding cards, badges, banners, and the bottom tab bar. All inline-styled in the codebase (no CSS framework), all built from --text, --card, --accent.

Buttons

Three roles. Primary drives the next step (one per view). Secondary is the dashboard chrome (Refresh, Settings). Tinted uses the accent at low-alpha for emphasis without primary weight (Manage). All buttons hit a 44 px minimum height.
